*** Man (manual) pages

You can access the *man page for a command* with either of:

#+BEGIN_EXAMPLE
git &lt;command&gt; --help
#+END_EXAMPLE

#+BEGIN_EXAMPLE
git help &lt;command&gt;
#+END_EXAMPLE

#+BEGIN_EXAMPLE
man git-&lt;command&gt;
#+END_EXAMPLE

Example:

#+BEGIN_src sh
git commit --help
#+END_src

#+BEGIN_src sh
git help commit
#+END_src

#+BEGIN_src sh
man git-commit
#+END_src

*** Command options

To get a *list of the options for a command*, type:

#+BEGIN_EXAMPLE
git &lt;command&gt; -h
#+END_EXAMPLE

Example:

#+BEGIN_SRC sh
git commit -h
#+END_SRC
